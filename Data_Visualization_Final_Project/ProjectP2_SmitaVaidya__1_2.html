def AgeGroup(row):
    if row['Age'] < 15:
        return 'Children'
    elif row['Age'] < 25:
        return "Youth"
    elif row['Age'] < 35:
        return "YoungAdult"
    elif row['Age'] < 50:
        return "MiddleAged"
    elif row['Age'] >= 50:
        return "ElderlyOld"
    
titanic_pd['Age_Group'] = titanic_pd.apply(AgeGroup, axis = 1)

Next, I am going to look at whether there are differences in survival rates across sex, passenger class, embarked port, family and age group. In order to do that I will define a couple of functions and use them across the different variables. Both of these functions calculate crosstabs for 2 or 3 variables for a given dataset. The cross tabulation of various variables against each other will help me see the breakdowns and relations between these variables.

Function to determine survival proportions for 1 variable¶

In [319]:
def CrossProp(df, Outcome, Variable):
    outcome_by_var = pd.crosstab(index = df[Outcome], 
            columns = df[Variable],
            margins = True)
    print "Proportion of %s by %s" %(Outcome, Variable), "\n"
    print outcome_by_var, "\n"
    print outcome_by_var/outcome_by_var.ix["All"]

Function to determine survival proportions for 2 variables¶

In [320]:
def CrossProp2(df, Outcome, Variable1, Variable2):
    outcome_by_var = pd.crosstab(index = df[Outcome], 
            columns = [df[Variable1],df[Variable2]],
            margins = True)
    print "Proportion of %s by %s and %s" %(Outcome, Variable1, Variable2), "\n"
    print outcome_by_var, "\n"
    print outcome_by_var/outcome_by_var.ix["All"]

Q9. Is there a statistically significant difference between average ages of male and female survivors?¶

  • Null hypothesis: average age of male survivors (Am) = average age of female survivors (Af)
  • Alternative hypothesis: Am != Af

First I will create a subset of the data that comprises only survivors. There are 342 survivors.

Male survivor ages:

Female Survivor Ages:

Number of female survivors is: 233

T-Test

  • The package scipy has a statistical subset stats, which has a function ttest_ind to check for this.
  • It needs 2 input arrays for male and female ages.
  • I will perform the test at an alpha value of 0.05 or a confidence of 95%. This means that the p-value needs to be less than 0.05 for the null hypothesis to be rejected.
  • The output of the test is of the format: (t-test, p-value)
In [321]:
import scipy.stats as stats
stats.ttest_ind(males_age, females_age)
Out[321]:
Ttest_indResult(statistic=-0.8350794934208513, pvalue=0.40425935251398093)

I see that the pvalue is 0.404, much higher than 0.05, and hence I cannot reject the null hypothesis at a significance level (alpha level) of 0.05 or 95% confidence.

Thus I conclude that there is no significant difference between mean ages of male and female survivors of the Titanic. However, I must keep in mind that my age data contains some artificial values (median age of 28 was inserted into missing data points) and so I do not want to make any strong conclusions.

Q10. Is there a statistically significant difference between average fares paid by survivors and non-survivors?¶

  • Null hypothesis: average fare paid by survivors (Fs) = average fare paid by non-survivors (Fns)
  • Alternative hypothesis: Fs != Fns

First I need to split the Titanic data into 2 datasets for survivors and non-survivors. I already have the survivors dataset from the previous questions and will reuse that.

Testing the hypothesis

  • I perform another independent 2 sample t-test using the scipy stats library as seen in the analysis for the previous question.
  • It needs 2 input arrays for survivor and non-survivor fares.
  • I will perform the test at an alpha value of 0.05 or a confidence of 95%. This means that the p-value needs to be less than 0.05 for the null hypothesis to be rejected.
  • The output of the test is of the format: (t-test, p-value)
In [295]:
stats.ttest_ind(survivors_fare, nonsurvivors_fare)
Out[295]:
Ttest_indResult(statistic=7.9391916608710549, pvalue=6.1201893419241981e-15)

As seen above, the p-value is very small, much lower than the t-critical value of 0.05 (95% confidence) and lower than even the t-critical value of 0.01 (99% confidence) if I wanted a higher confidence. I do not accept the null hypothesis and conclude that the alternate hypothesis is true.

Thus I can see that the fares paid by survivors of the Titanic were much different (in fact, much higher) than those paid by non-survivors.
